# DateDrop

An interactive, flirty date invite page built as a polished single-page experience.

## Experience

The app now focuses on one shareable invite flow at `/`:

- Cinematic welcome
- Flirty name entry
- Main question: "Will you go on a date with me?"
- Playful, non-frustrating "No" interaction
- Rewarding "Yes" celebration
- Date picker
- Time picker
- Food selection cards
- Vibe selection cards
- Final confirmation summary with replay and save/share actions

## Themes

The UI includes switchable themes from `ui-theme-design-library.md`:

- Soft Minimalism
- Swiss International Style
- Fashion Editorial
- Art Deco Digital
- Retro Futurism

## Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
